## Key Findings

- Workbook has 2 sheets: `KADAVANTHARA` and `EDAPPALLY`.
- Both sheets contain Advanced plan pricing blocks.
- Common class-level Advanced pricing is identical between branches for overlapping classes.
- Edappally has an extra row: `9 State` (present in both quarterly and 6/8 blocks).
- Kadavanthra includes additional Advanced blocks not present in Edappally sheet:
  - HSS subject-wise (`physics`, `chemistry`, `maths`) for quarterly and 6/8 plans

## Validation Summary

- Quarterly blocks are internally consistent: Q1+Q2+Q3+Q4 = Quarterly Total for all rows.
- 6-instalment blocks are internally consistent: (1st-5th * 5) + 6th Final = 6 Inst Full.
- 8-instalment blocks are internally consistent: (1st-7th * 7) + 8th Final = 8 Inst Full.

## Update-Ready Structure (Recommended)

Use the following operational structure while updating existing Advanced fees:

1. `advanced.class.quarterly` (branch + class)
2. `advanced.class.instalment_6_8` (branch + class)
3. `advanced.hss.quarterly` (branch + subject) [Kadavanthra present; confirm for Edappally]
4. `advanced.hss.instalment_6_8` (branch + subject) [Kadavanthra present; confirm for Edappally]
5. `advanced.class10_subject.quarterly` (branch + subject) [Kadavanthra present; confirm for Edappally]
6. `advanced.class10_subject.instalment_6_8` (branch + subject) [Kadavanthra present; confirm for Edappally]

Important: For Edappally, this workbook currently defines only class-level Advanced blocks. Subject-wise and HSS Advanced blocks must be explicitly confirmed before copying from Kadavanthra.
